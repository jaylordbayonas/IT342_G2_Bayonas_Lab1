# Backend API

Spring Boot REST API for user registration and authentication.

## Status
In Planning Phase

## Planned Endpoints
- `POST /api/auth/register` - User registration
- `POST /api/auth/login` - User login
- `GET /api/user/profile` - Get user profile
- `PUT /api/user/profile` - Update user profile

## Technology Stack
- Spring Boot
- Spring Security
- JPA/Hibernate
- MySQL/PostgreSQL
